# Variable Geometry Wing Optimization

This project simulates and optimizes a retractable variable-geometry wing system to minimize aerodynamic drag while maintaining lift and stability constraints.

## Features
- Aerodynamic modeling (lift and drag)
- Optimization of wing extension
- Flight condition simulation
- Future: closed-loop control system

## Motivation
Inspired by modern aerospace challenges including supersonic flight and adaptive aircraft geometry.

## Tech Stack
- Python
- NumPy / SciPy
- Matplotlib
- (Future) Machine Learning models
